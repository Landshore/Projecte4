# T02 — DRP: Cas Pràctic i Definició d’Estratègia

## Descripció
Aplicació pràctica dels conceptes DRP: definició d’un pla de recuperació complet per a un escenari plantejat pel professorat.

## Checklist
- [ ] Analitzar l’escenari proposat

- [ ] Definir periodicitat i retenció
- [ ] Dissenyar checklists de recuperació
- [ ] Validar coherència del pla
