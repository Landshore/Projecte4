# T01 — DRP: Anàlisi de Cas del Client

## Descripció
Anàlisi del cas real proporcionat. Identificació d’actius crítics, riscos, dependències, RTO/RPO i proposta d’una estratègia de recuperació.

## Checklist
- [ ] Llegir el cas del client en detall
- [ ] Identificar actius crítics (equips, servidors, dades)
- [ ] Identificar riscos i vulnerabilitats principals
- [ ] Definir paràmetres RTO/RPO coherents
- [ ] Plantejar estratègia de còpies de seguretat
- [ ] Redactar recomanacions finals
- [ ] Pujar informe al repositori
